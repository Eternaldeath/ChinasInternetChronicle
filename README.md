# 下载与使用

下载项目

```Git
git clone git@github.com:Eternaldeath/ChinasInternetChronicle.git
```

下载本地服务器

```javascript
npm install anywhere -g 
```

在项目根目录下，启用本地服务器

```javascript
anywhere 
```
